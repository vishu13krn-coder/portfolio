# Vishwas K Portfolio

Premium static portfolio website for Vishwas K, built with HTML5, CSS3, and vanilla JavaScript.

## Structure

- `index.html`
- `css/style.css`
- `css/animations.css`
- `css/responsive.css`
- `js/script.js`
- `js/particles.js`
- `assets/`

## Usage

Open `index.html` directly in a browser or deploy the `portfolio` folder to GitHub Pages.
